# Message Bubbles

Displays message bubbles as a conversation.

## Usage

```latex
% loading of the package
\usepackage{messagebubbles}

% usage
\messagebubbleleft{01.01.2011 14:23}{This is my text}
\messagebubbleright{01.01.2011 14:25}{This is another text}
```
